# B4 Project Dashboard

Browser-based visualization of project status.

## Features

### Project Overview
- Current phase indicator
- Milestone progress
- Key metrics (bugs, blockers, completion %)

### Sprint View
- Current sprint goals
- Task completion status
- Team workload

### Bug Tracker
- Open bugs by severity
- Bug trend over time
- Fix version tracking

### JIRA Integration
- Live ticket status
- Assignment overview
- Sprint burndown

### Confluence Links
- Quick access to key pages
- Recent updates

## Tech Stack
- HTML/CSS/JavaScript (vanilla, no framework)
- Fetches data from JSON files generated by summary agents
- Auto-refreshes on interval

## Usage
```bash
# Generate dashboard data
python3 ../summary/status_report.py --output data/status.json
python3 ../summary/bug_tracker.py --output data/bugs.json

# Serve dashboard
python3 -m http.server 8080
# Open http://localhost:8080
```

## Files
- `index.html` - Main dashboard page
- `data/` - JSON data files (gitignored, generated)
- `style.css` - Dashboard styling
